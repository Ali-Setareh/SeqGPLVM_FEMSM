# NonLinear_FEMSM
This repository contains the code and experiments for my Master’s thesis, where I extend recent work on causal inference in longitudinal panel data by combining Sequential Gaussian Process Latent Variable Models (SeqGPLVM) (Hatt &amp; Feuerriegel, 2024) with Fixed-Effect Marginal Structural Models (FE-MSMs) (Blackwell &amp; Yamauchi, 2024).
